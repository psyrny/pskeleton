My nette sandbox
=======================

Installation
--------------
###Clone:
	$ git clone git@github.com:psyrny/pskeleton.git

###Install npm packages
	$ npm install

###Install php dependencies
	$ composer update

###Install bower components
	directory: /www/...bower.json
	packages are installed to: /www/src/bower_components/...
	$ bower install

###Run gulp
	$ gulp watch
